# La Previa 🍻

Juego de fiesta para previas y quedadas con amigos. Web, sin instalar nada: se abre en el navegador y se comparte por link.

## Modos de juego

- **🃏 Retos** — 4 intensidades (😇 Suave · 🌶️ Picante · 🔥 Salvaje · 😈 Bestia) + **Modo Noche** que sube solo según las copas que lleváis.
- **🪙 La Moneda** — preguntas "¿quién es más...?" con la mecánica de señalar en secreto y dejar que la moneda decida si se revela. Dos estilos: **Normal** (cualquier grupo) y **Mixto** (grupos con chicos y chicas, +18).

## Cómo se juega

1. Añade los nombres del grupo.
2. Elige juego, intensidad/estilo y dale a Empezar.
3. Pasaos el móvil y a por la noche. 🌙

## Tecnología

HTML + CSS + JavaScript puro. Sin dependencias. Contenido pre-generado (cero coste por partida).
